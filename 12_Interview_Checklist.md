# Interview Preparation Checklist

## Technical Review (Do This First)

### SQL - Must Know
- JOINs (INNER, LEFT, RIGHT, FULL, SELF, CROSS)
- Subqueries vs CTEs
- Window functions (ROW_NUMBER, RANK, DENSE_RANK, LAG, LEAD)
- GROUP BY with HAVING
- CASE statements
- Date functions
- Indexes and optimization basics

### Excel - Must Know
- VLOOKUP/XLOOKUP/INDEX-MATCH
- PivotTables and PivotCharts
- SUMIFS/COUNTIFS/AVERAGEIFS
- IF/IFS with nested logic
- Data validation and conditional formatting
- Power Query basics
- Basic charts and dashboards

### Power BI - Must Know
- Data modeling (star schema)
- DAX measures vs calculated columns
- CALCULATE and filter context
- Time intelligence functions
- Basic visualizations
- Relationships and cardinality

### Statistics - Must Know
- Mean, median, mode - when to use each
- Standard deviation and variance
- Correlation vs causation
- Hypothesis testing basics (p-value)
- A/B testing fundamentals

## Day Before Interview
- Review your resume - know every project
- Prepare 3-5 STAR stories
- Research company thoroughly
- Prepare questions to ask
- Get good sleep

## Day Of
- Arrive 10-15 minutes early (or test tech for virtual)
- Bring: resume, notebook, pen, portfolio (if applicable)
- Dress professionally
- Stay calm, think aloud, ask clarifying questions
